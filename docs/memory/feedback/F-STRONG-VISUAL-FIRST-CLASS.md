# Strong visual = first-class axis

战友把强视觉作为 ScoutFlow **一级 axis** (跟最大马力 / 最高质量 / 第一性原理同级), 不是 nice-to-have. H5 Capture Station 不许变 generic admin / dashboard 气质. 5 Gate aesthetic checklist (视觉层级 / 空间对齐 / 遮挡安全 / 字体可读 / 视觉重量) 是交付硬门, V-PASS 由战友浏览器 review 拍板.

**Why:** 战友审美红线明示拒绝 generic admin shell. ScoutFlow `~/.claude/rules/aesthetic-first-principles.md` 已锁 5 Gate. 历史 PR #122 PF-V (visual prototype lane) / PR #243 PF-C4-01 (13 surface workstation shell) 都是这条 axis 的 enforce.

**How to apply:** 任何 surface PR 必跑 5 Gate self-audit. commander prompt 必含 motion vocab + tokens.css 三层 overlay + 自写 SVG sprite + capture-side motion 边界. 见到"先实现再美化"反 — 视觉是输入, 不是 polish 环节.
