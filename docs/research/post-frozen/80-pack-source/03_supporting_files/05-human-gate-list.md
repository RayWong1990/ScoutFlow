# Human Gate List

| Gate | Required before | Minimum decision |
|---|---|---|
| URL_selection | C1 real URL proof | 3 canary URLs |
| usefulness_verdict | C1 pass | at least 2/3 useful or actionable |
| raw_handoff_permission | C2 manual handoff | user agrees to manual transfer path |
| script_seed_acceptance | C2 pass | at least one downstream paragraph/script seed accepted |
| visual_verdict | C4 visual pass | human says visual surface is usable, not merely technically rendered |
| true_write_approval | any true vault write | explicit separate approval after proof |
| explicit_runtime_approval | BBDown/ASR/browser/migration | explicit separate approval and dispatch |
