# Manual RAW Handoff Runbook

## Goal

Move a ScoutFlow preview-only candidate note into RAW `00-Inbox` manually, without enabling ScoutFlow true write.

## Steps

1. Open `docs/research/post-frozen/raw-handoff-staging/`.
2. Review `README-manual-transfer.md`.
3. Copy the staged `.md` notes into `~/workspace/raw/00-Inbox/`.
4. In RAW, treat the files as intake candidates, not accepted final notes.

## Boundary

- ScoutFlow stops at repo-local staging.
- The user performs the actual copy.
- RAW remains the intake owner after the copy.
