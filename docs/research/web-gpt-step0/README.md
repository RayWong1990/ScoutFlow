# Web GPT STEP0 Templates

## 1. Purpose

This directory holds GitHub-visible authoring references for future web GPT runs.

It exists because web GPT can read GitHub App truth directly, and later prompts may need:

- a successor-pack authoring template
- a localhost-preview dispatch template
- a proof/overflow dispatch template

These files are candidate references only. They are not authority, not execution approval, and not runtime approval.
