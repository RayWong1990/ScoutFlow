# ScoutFlow Retro

`decision-log = forward decisions`

`retro = small retrospective scaffold`

本目录只承载四类最小回溯动作：

- weekly retro
- decision reversal
- scope shrink
- meta-governance learning

使用边界：

- not authority
- not PRD/SRD amendment
- not product scope

写作约束：

- 每次只写最小事实与结论
- 单次填写以 `15-minute max` 为上限
- retro 写完后，如需进入主线判断，再由 `docs/decision-log.md` 或 `docs/current.md` 做 authority writeback
