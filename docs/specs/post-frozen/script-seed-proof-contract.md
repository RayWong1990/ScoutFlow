# Script Seed Proof Contract

## Purpose

Define the minimum bar for saying a staged note produced a downstream script seed.

## Minimum Proof

A script seed must include:

1. one coherent paragraph
2. a clear subject
3. an explicit link back to the staged note / capture
4. wording that remains candidate-level, not final copy

## Non-Examples

- a filename only
- a title only
- a bullet list with no paragraph shape
- a paragraph that claims RAW acceptance when transfer has not yet happened
