# H5 Five Gate Checklist

## Gate 1 Visual Hierarchy

- signal: primary action and primary data surface are visually dominant
- evidence: pending screenshot or reviewer note
- status: candidate

## Gate 2 Spatial Alignment

- signal: key blocks align to a stable grid with no accidental drift
- evidence: pending screenshot or reviewer note
- status: candidate

## Gate 3 Occlusion Safety

- signal: no text, badge, or control overlaps another UI layer
- evidence: pending screenshot or reviewer note
- status: candidate

## Gate 4 Type Readability

- signal: headings, labels, and body copy remain legible at the intended viewport
- evidence: pending screenshot or reviewer note
- status: candidate

## Gate 5 Visual Weight

- signal: dense and light regions feel balanced; no panel overwhelms the shell
- evidence: pending screenshot or reviewer note
- status: candidate
